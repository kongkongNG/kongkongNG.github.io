LeetCode 820.单词的压缩编码

发表于 2020-03-28 更新于 2020-03-29 分类于 LeetCode 阅读次数： Valine：

题目描述

标准解法 字典树Trie $O(\Sigma wi)$

本题的核心就是如何找到是其他单词后缀的单词并删去。利用字典树，反向插入单词。这样后缀相同的单词在同一棵子树中，巧妙地过滤了需要删去的单词。最后只需要统计叶子所有叶子节点的高度，即为所求。
